# TypeScript Assignment

This repository contains the TypeScript modules for the assignment.

## Structure

```
typescript/
├── modulo-1/        # Module 1 — (description here)
│   ├── src/         # TypeScript source files
│   ├── dist/        # Compiled JavaScript output
│   └── docs/        # Module documentation
├── modulo-2/        # Module 2 — (description here)
│   ├── src/         # TypeScript source files
│   └── docs/        # Module documentation

```

## Getting Started

Navigate into any module and install dependencies:

```bash
cd modulo-1
npm install
```

Compile TypeScript:

```bash
npm run build
```

Run in watch mode during development:

```bash
npm run dev
```
